### About the project

A password manager.

### Built with

Django REST Framework and ReactJS.

[![React][React.js]][React-url]

[![Django][Django.py]][Django-url]

### Usage

1. Install dependencies
    ```sh
    pip install -r requirements.txt
   ```
2. Compose docker
    ```sh
    docker compose up -d
   ```
3. Run server
    ```sh
    python manage.py runserver
    ```
